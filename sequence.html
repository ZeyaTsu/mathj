Suites

Pré-requis - Vocabulaire

Le nombre Un est appelé le terme de rang n. - Exemple: Rang 1 =

$$U_1$$

I - Suites explicites et Suites par récurrence

Une suite explicite est définit comme une suite où pour tout terme de rang n, est calculable sans la nécessité du résultat de son rang précédent.

Ainsi pour une suite explicite nous avons comme exemple:

$$Un = 3n² + 1$$

Une suite par récurrence est définit comme une suite où pour tout terme de rang n, est nécessaire d'avoir le résultat de son rang précédent afin d'être calculable.

Ainsi pour une suite par récurrence nous avons comme exemple:

$$Un_+1 = Un + 3$$

II - Suite arithmétique

On dit qu'une suite est arithmétique si il existe un nombre r tel que, pour tout n :

$$U_n = U_0 + n*r$$

Ainsi, pour prouver qu'une suite est arithmétique, la différence doit être égal à une constante (r).

$$Un_+1 - Un = r$$

Ainsi, il est possible de trouver la raison r d'une suite arithmétique via une équation si l'on possède au moins le résultat de Un d'un autre

Sachant que

$$Un = U_0 + n*r$$

Alors

$$\frac{U_n - U_0}{n - 0} = r$$

Alors

$$U_0 = U_n -n*r$$

Afin d'étudier les variations d'une suite arithmétique:
Si r > 0 la suite Un est croissante.
Si r < 0 la suite est décroissante
Si r = 0 la suite est constante.

Pour calculer la somme des termes consécutifs d'une suite arithmétique:


$$\frac{(U_0 + U_n)*(n+1)}{2}$$

Et si la suite des termes est logique comme: 1 + 2 + 3 + ... + n = S, Alors


$$\frac{n*(n+1)}{2}$$

III - Suite géométrique

On dit qu'une suite est géométrique si il existe un nombre q tel que, pour tout n:

$$U_n = U_0 * q^n$$

Ainsi, pour prouver qu'une suite est géométrique, le quotient doit être égal à une constante (q).

$$\frac{Un_+1}{Un} = q$$

Ainsi, il est possible de trouver la raison r d'une suite arithmétique via une équation si l'on possède au moins le résultat de Un d'un autre

Sachant que

$$U_n = U_0 * q^n$$

Alors

$$(\frac{U_n}{U_0})^\frac{1}{n} = q$$

Alors

$$U_0 = \frac{Un}{q^n}$$

Si q > 1 alors la suite est croissante

Si 0 < q < 1 alors la suite est décroissante

Si q < 0 alors la suite est monotone

Si q = 1 alors la suite est constante

Pour calculer la somme des termes consécutifs d'une suite géométrique:


$$U_0*\frac{1-q^{n+1}}{1-q}$$
